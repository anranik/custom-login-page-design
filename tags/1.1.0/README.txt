== Description ==

This plugin will help you to modify your admin page. Its very easy to customize login page with this.
You can add login page background color, background image, color of the links,button etc.


All can be done with this:

*   Upload custom logo to wp-admin
*   Change background color of login page
*	Add color to format
*	Add background to login page
*	Change the color of submit button
*	Change te color of links on wp-login form


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-login-customize.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Find the settings under `Settings->Wp Login Customze`
